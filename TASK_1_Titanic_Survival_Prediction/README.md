# Task-1: Titanic Survival Prediction

## Description
This project predicts passenger survival on the Titanic using Python. It covers data preprocessing, exploratory data analysis (EDA), model training, and evaluation.

## Files Included
- **Task-1/CODSOFT_Task1_Titanic_Survival_Prediction.ipynb** → Jupyter notebook containing all steps and code.
- **Task-1/Titanic-Dataset.csv** → Dataset used for prediction.
- **Task-1/Images/** → Folder containing output images including:
  - Model evaluation
  - Confusion matrix
  - Survival predictions

## Output Highlights
- **Model accuracy:** See `Task-1/Images/model_evaluation.png`
- **Confusion matrix:** See `Task-1/Images/confusion_matrix.png`
- **Survival predictions:** See `Task-1/Images/survival_prediction.png`

## Instructions to Run
1. Open the notebook in **Jupyter Notebook** or **VS Code**.
2. Ensure `Titanic-Dataset.csv` is in the same folder as the notebook.
3. Run all cells to reproduce the results.

## Notes
- Make sure all required libraries (pandas, numpy, matplotlib, seaborn, scikit-learn) are installed.
- You can customize or extend the models to improve accuracy.
